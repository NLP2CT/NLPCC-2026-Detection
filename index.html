ShareTask

一个更清晰的静态站点入口

这个页面用于演示并启动 GitHub Pages 自动部署。你把真实的 ShareTask 静态页面文件放入仓库根目录后， 会由 Actions 自动更新线上内容。

如何更新页面

直接修改/替换仓库根目录的 `index.html`（以及相关静态资源：`css/`、`js/`、`img/`）。

自动部署

推送到 `main` 后会触发 workflow：`.github/workflows/pages.yml`，部署到 GitHub Pages。

下一步

把你的 `sharetask` 静态网站内容拷贝进来即可。需要更复杂的构建流程（如 `npm run build`）也可以再扩展。

如果你看到 404：先确认仓库 `Settings -> Pages` 已启用为 `GitHub Actions`，再等待一次 Actions 部署完成。
